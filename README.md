## Overview

Gradient Generator is a simple frontend project that generates random linear gradients in real time. Users can instantly preview the generated gradient and copy the corresponding CSS code for use in their own projects.

---

## Features

- Random gradient generation
- Live gradient preview
- Copy CSS code to clipboard
- Responsive interface
- Lightweight and fast
- Beginner-friendly project
